🌍 GIWPS Conflict Tracker

A lightweight, no-API-key web app to identify and analyze recent news and reports related to **Gender, Peace, and Security (WPS)** across countries.

Built using Python and Streamlit.

---

What this tool does

This app helps researchers, policymakers, and students:

- Search for **recent (last 30 days or custom range)** sources
- Focus on:
  - Women, girls, boys, children
  - Gender-based violence (GBV)
  - Sexual violence / CRSV
  - Women, Peace and Security (WPS) agenda
  - National Action Plans (NAPs)
  - Conflict impacts and displacement
  - Climate and environmental security
- Automatically:
  - Collect sources from trusted public sites
  - Filter by country and keywords
  - Score and tag results
  - Display structured outputs

---

Data sources (no API required)

The tool pulls from curated public sources such as:

- ReliefWeb  
- UN News  
- UN Women  
- UNICEF  
- International Crisis Group  
- Human Rights Watch
